# Archived Milestone Batches

**Purpose:** Define the scope of absorbed or historical milestone batches under `docs/archive/milestones/`; these files are provenance snapshots, not default maintenance input.

**Last updated:** 2026-04-16

## Scope

1. Store milestone batches whose durable rules have already been lifted into active docs.
2. Preserve frozen acceptance wording for audit and historical comparison.
3. Keep current active milestone batches in `../../milestones/`.

## Rules

1. Do not move a milestone batch here until active docs no longer require it as default reading.
2. When archiving a batch, update any active links that still point to it as if it were current.
3. Use these files only when exact historical acceptance wording is required.
