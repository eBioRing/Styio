# Archived Rollups

**Purpose:** Define the scope of archived provenance digests under `docs/archive/rollups/`; these files keep compressed historical summaries after their durable maintenance rules have been absorbed into active SSOTs and runbooks.

**Last updated:** 2026-04-16

## Scope

1. Store compressed historical summaries that are still useful for provenance, but no longer belong in the active `docs/rollups/` cold-start path.
2. Keep active maintenance summaries in `../../rollups/` and move absorbed historical digests here once they stop being required for normal maintenance.
3. Treat these files as optional background only; active design, spec, runbook, and workflow docs remain the default read chain.
