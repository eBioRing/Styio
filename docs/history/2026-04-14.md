## Checkpoint（D.6/F.4：Nightly Parser Shadow Fallback 修复 + 全绿综合基线）

### 修复内容

- `src/StyioParser/NewParserExpr.cpp`
  - nightly statement route 现已覆盖 `->` 与 `~>` 路径，`t07_redirect` 和 `t08_arbitrage` 不再因为 token gate 缺口落回 legacy。
